# Las Higueras Activa 🏙️
### Gestión Ciudadana Modernizada - Un puente directo entre el Vecino y el Municipio

---

## 🔝 Propuesta de Valor
**Las Higueras Activa** es una plataforma integral diseñada para transformar la relación entre la Municipalidad y sus fuerzas vivas (vecinos, comercios e instituciones). No es solo una app de reclamos; es una **herramienta de gestión de datos en tiempo real** que permite al Intendente y a sus equipos operativos reaccionar con precisión, transparencia y trazabilidad.

---

## 🚀 Funcionalidades Clave

### 👨‍👩‍👧‍👦 Para el Ciudadano (App Móvil)
*   **Reportes en un clic:** Reporte de problemas (alumbrado, residuos, calles) con geolocalización GPS precisa y evidencia fotográfica.
*   **Identificación Estratégica:** Perfiles diferenciados para un vecino común, un comercio local o una institución pública (escuelas, centros de salud).
*   **Transparencia Total:** Seguimiento del estado de su trámite en tiempo real desde el celular.
*   **Buzón Oficial:** Recibe avisos críticos y novedades del municipio de forma directa y unidireccional.

### 🏛️ Para la Municipalidad (Panel de Control)
*   **Mapa Geo-referenciado:** Visualización de calor sobre los problemas de la ciudad (rojo: pendiente, amarillo: en proceso, verde: resuelto).
*   **Gestión Operativa:** Capacidad de asignar responsables y agregar notas técnicas que el vecino recibe al instante.
*   **Métricas de Gestión:** Seguimiento de la tasa de resolución y tiempos promedio de respuesta para evaluar el desempeño municipal.
*   **Numeración Inteligente:** Sistema de tickets `YYMMDDXXXX` para métricas mensuales y orden administrativo.

---

## 🛠️ Stack Tecnológico
*   **Frontend:** Next.js 15+ (App Router)
*   **Estilos:** Tailwind CSS v4 con estética "Glass-morphism" (moderna y limpia)
*   **Animaciones:** Framer Motion (para una experiencia premium y fluida)
*   **Geolocalización:** Web Geolocation API para precisión territorial.
*   **Persistencia:** Gestión de estado local y localstorage para simulación de base de datos.
